# 配置文件说明

| 文件 | 是否提交 Git | 说明 |
|------|-------------|------|
| `bots.yaml.example` | 是 | 台账模板，复制为 `bots.yaml` |
| `bots.yaml` | **否** | 你的被测 Bot 真实配置 |
| `bots_registered.yaml.example` | 是 | 空台账模板 |
| `bots_registered.yaml` | **否** | 群内注册自动写入 |
| `test_defaults.yaml` | 是 | 项目级默认话术/文档链接（可改占位符） |
| `test_cases.yaml` | 是 | 用例定义 |
| `environments.yaml` | 是 | 超时档位与环境 |

首次部署步骤见 [docs/GETTING_STARTED.md](../docs/GETTING_STARTED.md)。
